# 🚀 AI Travel India — Futuristic 3D Travel Platform

A next-generation **AI-powered travel landing page** built with modern web technologies.  
This project delivers a **futuristic, immersive experience** using 3D visuals, smooth animations, and intelligent UI design.

---

## 🌍 Live Concept

> Explore India like never before with AI-powered trip planning and immersive UI.

---

## ✨ Features

### 🎨 UI / UX

- Futuristic dark theme with neon glow
- Glassmorphism design system
- Fully responsive layout
- Smooth scroll animations

### 🌐 3D Experience

- Interactive 3D globe using Three.js
- Animated starfield background
- Floating and motion-based elements

### 🤖 AI Experience (UI Level)

- AI trip planner section
- Smart input-based trip generation (UI)
- AI presence indicators

### 📦 Sections Included

- Hero (3D + CTA)
- Destinations
- AI Trip Planner
- Experience Categories
- Testimonials
- Call-To-Action
